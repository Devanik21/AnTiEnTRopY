# Contributing to AnTiEnTRopY

Thank you for your interest in contributing to AnTiEnTRopY! We welcome contributions from the community.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally: `git clone https://github.com/your-username/AnTiEnTRopY.git`
3.  **Create a new branch** for your feature or bugfix: `git checkout -b your-feature-branch`
4.  **Make your changes** and commit them with descriptive messages.
5.  **Run tests** (if applicable) to ensure your changes don't break existing functionality.
6.  **Push your branch** to your fork: `git push origin your-feature-branch`
7.  **Submit a Pull Request (PR)** to the main repository.

## Submitting Issues

If you find a bug or have a feature request, please open an issue on GitHub. Try to be as detailed as possible, providing steps to reproduce for bugs and clear use cases for feature requests.

## Code Style

Please adhere to standard Python formatting guidelines (e.g., PEP 8). We recommend using `black` and `flake8`.

Thank you for contributing!
